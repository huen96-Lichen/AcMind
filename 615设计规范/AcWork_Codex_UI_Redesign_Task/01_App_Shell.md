# 01 统一应用外壳

## 目标

统一应用外壳负责提供稳定导航、页面上下文和窗口层级。它不能抢占内容，也不能在不同页面中改变基础结构。

## 基准尺寸

- 设计画布：1500 × 920 pt
- 最小窗口：1180 × 720 pt
- 主侧边栏：216 pt
- 顶部工具栏：60 pt
- 页面边距：20 pt
- 区块间距：16 pt
- 标准 Inspector：304 pt
- 标准 Filter Rail：220 pt

## 结构

```text
RootWindow
├── Sidebar 216
└── Workspace
    ├── Toolbar 60
    └── PageBody
        ├── Optional FilterRail
        ├── MainContent
        └── Optional Inspector
```

## 主侧边栏

### 分组

工作：
- 工作台
- Agent
- 收集箱
- 日程

处理：
- 工具台

随身能力：
- 灵动大陆
- 说入法

系统：
- 状态
- 模型
- 设置

### 规格

- 品牌区高度：72
- 导航项高度：38
- 导航项圆角：10
- 图标：16–18 pt
- 分组标题：11 pt semibold
- 选中状态：浅蓝背景 + 系统蓝图标
- Hover：浅灰背景
- 不显示永久快捷键标签，Hover 时可显示
- 底部显示本地服务状态与当前模型

## 顶部工具栏

只展示当前页面上下文。

左侧：
- 页面标题
- 当前项目、筛选条件或日期范围

右侧：
- 页面主要操作
- 搜索
- 快速记录
- 必要状态

页面标题不得在正文重复。

## 响应式

- ≥1320：允许三栏
- 1180–1319：隐藏 Inspector，详情改为 Sheet 或 Overlay
- <1180：禁止继续缩小
- 高度 <760：收紧垂直间距，但不隐藏核心操作

## 窗口层级

允许阴影的组件：
- Popover
- Sheet
- Context Menu
- Floating Input
- 灵动大陆
- HUD

常规卡片不使用明显阴影，只使用 1 pt 描边。

## Codex 验收

- 11 个页面全部复用同一个 Shell
- Sidebar 不因页面切换改变宽度
- Toolbar 高度固定
- 页面标题不重复
- 1180 × 720 无裁切
- 键盘焦点顺序正确
