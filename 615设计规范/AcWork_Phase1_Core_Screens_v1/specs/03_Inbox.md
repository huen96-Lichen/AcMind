# 03 收集箱

## 页面定位

收集箱是所有内容进入 AcWork 后的统一浏览、整理和分发中心。

剪贴板、手机同步、说入法、截图、Agent 和手动添加都属于内容来源，不再拥有独立一级页面。

## 三栏布局

```text
FilterRail 220
ContentArea 1fr
Inspector 304
```

## 左侧：来源与筛选

### 快捷视图

- 全部内容
- 待整理
- 已 Pin
- 已收藏
- 最近使用

### 来源

- 剪贴板
- 手机同步
- 说入法
- 截图与 OCR
- Agent
- 手动添加

### 类型

- 文本
- 链接
- 图片
- 文件
- 代码
- 富文本
- 视频

### 状态

- 待整理
- 已提炼
- 已归档
- 已导出

### 规格

- 分组标题 11 pt semibold
- 筛选项高 34
- 支持多条件组合
- 当前条件在顶部工具栏下方显示 Filter Chips
- 每个筛选项可显示数量

## 中间：内容工作区

### 顶部操作条

- 当前筛选摘要
- 搜索
- 排序
- 列表 / 网格切换
- 批量选择
- 添加内容

### 列表视图

每行：

- 类型图标
- 标题
- 两行预览
- 来源应用或设备
- 创建时间
- 状态
- 标签
- Pin
- 收藏
- 更多

规格：

- 标准行高 84
- 紧凑行高 64
- 选中状态使用极浅蓝背景
- Hover 显示快捷操作

### 网格视图

卡片：

- 宽度 240–300
- 高度 168–196
- 内容本体占主要区域
- 底部固定元信息
- 不显示重复的大标题

不同类型采用不同预览：

- 图片：图片预览
- 链接：站点图标 + 标题 + 摘要
- 代码：等宽字体 + 语言标签
- 文件：文件图标 + 名称 + 大小
- 文本：最多 5 行

## 右侧：详情与处理

### 无选中项

显示：

- 总数量
- 待整理数量
- Pin 数量
- 手机同步状态
- 粘贴队列数量

### 有选中项

显示：

- 完整内容
- 来源
- 来源应用 / 设备
- 创建时间
- 类型
- 标签
- 关联项目

AI 操作：

- 自动标题
- 摘要
- 提取待办
- 提取日程
- 润色
- 发送给 Agent

去向：

- 转任务
- 添加到日程
- 保存到知识库
- 导出 Markdown
- 归档
- 删除

底部固定主要操作区。

## 剪贴板专项能力

当来源筛选为“剪贴板”时：

- 顶部显示监听状态
- 支持暂停监听
- 显示来源应用
- 支持复制
- 支持加入粘贴队列

## Pin 管理

Pin 是所有收集项的通用能力，不只属于剪贴板。

支持：

- 显示全部 Pin
- 隐藏全部 Pin
- 关闭全部 Pin
- 调整顺序

## 粘贴队列

- 仅队列非空时显示
- 支持拖动排序
- 单项移除
- 清空
- 连续粘贴

## 批量操作

- 添加标签
- 发送给 Agent
- 转任务
- 转日程
- 归档
- 导出
- 删除

## 响应式

宽度低于 1320：

- Inspector 自动隐藏
- 点击内容后使用右侧 Sheet
- FilterRail 保持
- 网格最小两列

## Codex 验收

- 剪贴板不再出现为一级页面
- 所有来源使用统一数据模型
- 支持列表和网格
- 支持组合筛选
- 支持批量操作
- 支持无选中和有选中两种 Inspector 状态
- 支持手机同步、Pin 和粘贴队列
