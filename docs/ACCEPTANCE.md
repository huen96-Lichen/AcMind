# 验收标准

## 核心链路验收

### 1. 采集流程

| 测试项 | 验收标准 |
|--------|---------|
| 手动输入 | 输入文本 → 生成 SourceItem → 状态为 inbox |
| 剪贴板监听 | 复制文本 → 自动采集 → 出现在收件箱 |
| 文件拖拽 | 拖入 PDF/DOCX/TXT → 解析内容 → 生成 SourceItem |
| 网页采集 | 输入 URL → 抓取正文 → 生成 SourceItem |
| 截图采集 | 触发截图 → OCR 识别 → 生成 SourceItem |

### 2. AI 蒸馏

| 测试项 | 验收标准 |
|--------|---------|
| Mock 蒸馏 | 无 AI 配置时 → 使用规则蒸馏 → 生成结构化输出 |
| 本地蒸馏 | Ollama 配置 → 发送请求 → 返回 JSON 结果 |
| 云端蒸馏 | OpenAI API 配置 → 发送请求 → 返回 JSON 结果 |
| 分层路由 | 根据内容类型 → 自动选择模型层级 |
| 重试机制 | 请求失败 → 自动重试 → 最终记录错误 |

### 3. 导出到 Obsidian

| 测试项 | 验收标准 |
|--------|---------|
| 单条导出 | 选择内容 → 导出 → Vault 中生成 .md 文件 |
| Frontmatter | 导出文件包含 title/tags/source/date 等元数据 |
| 批量导出 | 多选 → 批量导出 → 每条独立文件 |
| 冲突处理 | 同名文件 → 自动重命名（_1, _2...） |
| 路径规则 | 按日期/标签分类 → 正确的目录结构 |

### 4. 搜索

| 测试项 | 验收标准 |
|--------|---------|
| 关键词搜索 | 输入关键词 → 返回匹配的 SourceItem |
| 标签筛选 | 选择标签 → 过滤结果 |
| 空结果 | 无匹配 → 显示空状态 |

## UI 验收

### 5. 页面加载

| 页面 | 验收标准 |
|------|---------|
| 收件箱 | 加载后显示 SourceItem 列表，不白屏 |
| 蒸馏页 | 加载后显示蒸馏结果，不白屏 |
| 导出页 | 加载后显示导出列表，不白屏 |
| 设置页 | 加载后显示设置表单，不白屏 |
| 搜索页 | 加载后显示搜索框，不白屏 |
| 历史页 | 加载后显示历史记录，不白屏 |
| 知识库 | 加载后显示知识卡片，不白屏 |
| AI 工作台 | 加载后显示工作台，不白屏 |

### 6. ErrorBoundary

| 测试项 | 验收标准 |
|--------|---------|
| 页面级兜底 | 单页面崩溃 → 显示错误 UI → 侧边栏仍可导航 |
| 全局兜底 | 根组件崩溃 → 显示错误 UI + 重试按钮 |
| 错误上报 | 页面崩溃 → 错误记录到 error_log |
