# AcMind 产品介绍

> **个人桌面 AI 信息中枢**
> 把零散信息收集起来，整理成真正有价值的 Markdown 知识。

---

## 产品定位

每天我们都会遇到大量碎片化信息——一段文字、一张截图、一个网页链接、一段语音备忘。它们散落在剪贴板、聊天记录、浏览器标签页里，很难系统化管理。

**AcMind** 是一款 macOS 桌面应用，它做的事情很简单：

```
收集 → 暂存 → 整理 → 审阅 → 导出
```

把所有零散信息统一收集，用 AI 自动提炼成结构化知识，最终以 Markdown 格式沉淀到你的 Obsidian Vault 或本地目录。

---

## 核心功能

### 1. 多源收集

AcMind 支持从多种来源捕获信息，覆盖日常工作中绝大多数信息类型：

| 来源 | 方式 | 说明 |
|------|------|------|
| 手动文本 | 输入框直接输入 | 快速记录想法、灵感 |
| 剪贴板 | 自动监控 | 复制即收集，无需额外操作 |
| 截图 | 全屏 / 固定尺寸 / 区域选择 | 系统级快捷键触发 |
| 网页 | URL 粘贴 | 保存链接和页面信息 |
| 文件导入 | 拖拽 / 选择文件 | PDF、DOCX、图片、音视频 |
| 语音 | 录音转写 | 语音备忘自动转文字 |

**桌面胶囊**（Capsule）提供常驻的快速收集入口，随时唤起，一键捕获。

---

### 2. 暂存池

所有收集的内容统一进入暂存池，作为整理前的缓冲区：

- **统一视图**：所有来源的内容在一个列表中展示
- **快速预览**：选中即可查看内容详情
- **分类筛选**：按类型（文本/图片/截图/文件/网页/音频）快速过滤
- **搜索**：支持标题和内容全文搜索
- **批量操作**：一键送入整理，或批量删除

暂存池的设计理念是"先收集，不遗漏"，让用户零负担地积累素材。

---

### 3. AI 整理（蒸馏）

这是 AcMind 的核心能力。将暂存池中的内容送入整理流程后，AI 会自动进行：

- **标题提取**：为内容生成简洁准确的标题
- **摘要生成**：提炼核心要点
- **分类标注**：自动识别内容类别
- **标签提取**：生成语义化标签
- **正文整理**：将原始内容重新组织为结构化 Markdown

整理过程有清晰的进度反馈，列表项会显示 spinner 动画和状态标识。如果 AI 调用失败，会显示明确的错误信息和重试入口。

**AI 模型支持**：

- **本地模型**：通过 Ollama 接入，数据不出本机
- **云端 API**：支持 OpenAI 兼容接口（如 DeepSeek、通义千问等）
- **自动降级**：模型不可用时自动回退到规则模板引擎

---

### 4. 人工审阅

AI 整理完成后，结果进入审阅环节，用户可以：

- **预览**：查看 AI 生成的标题、摘要、标签和 Markdown 正文
- **编辑**：直接修改标题、摘要、标签，所有字段支持内联编辑
- **确认**：认可 AI 结果，进入导出流程
- **拒绝**：退回重新整理
- **确认并导出**：一步完成审阅和导出

审阅页面支持批量操作——全部确认或全部拒绝，适合处理积压内容。

---

### 5. Markdown 导出

确认后的内容可以导出为标准 Markdown 文件：

- **Frontmatter 规范**：自动生成包含标题、标签、日期、来源等元数据的 YAML Frontmatter
- **Obsidian Vault 对接**：直接写入配置的 Vault 目录，无缝融入已有知识体系
- **冲突处理**：文件名冲突时支持覆盖、跳过或自动重命名
- **原子写入**：使用临时文件 + 重命名机制，确保写入安全
- **导出记录**：完整的导出历史和状态追踪

导出的 Markdown 文件示例：

```markdown
---
title: "深度学习在自然语言处理中的应用"
tags: [AI, 深度学习, NLP]
category: "技术"
created: 2026-05-05
source: clipboard
---

# 深度学习在自然语言处理中的应用

## 摘要

本文介绍了深度学习技术在 NLP 领域的核心应用...

## 正文

（AI 整理后的结构化内容）
```

---

### 6. 知识库

导出的知识以知识卡片形式管理：

- **搜索**：支持全文搜索和标签过滤
- **分类浏览**：按类别查看知识卡片
- **来源追溯**：每张卡片可追溯到原始收集内容

---

### 7. 工具台

集成实用工具，无需离开 AcMind：

- **文件转换**：PDF、DOCX、图片等格式互转
- **OCR 识别**：图片文字提取
- **Agent 对话**：与 AI 助手交互
- **定时任务**：自动化重复操作

## 产品特点

**本地优先**：所有数据存储在本地，AI 处理可选本地模型，隐私可控。

**克制设计**：Apple 风格界面，大留白、低噪音，一个页面一个焦点，不做复杂后台管理。

**端到端打通**：从收集到导出的完整链路已贯通，每个环节都有明确的状态反馈和错误处理。

**灵活配置**：支持多种 AI 模型、自定义导出路径、个性化捕获设置。
