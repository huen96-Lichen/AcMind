# 人工最小维护模板

没有 AI 时，用户可以直接复制这个模板手动维护。

```markdown
---
schema_version: "0.2"
title: "这里写标题"
summary: "这里写一句话总结"
tags: []
category: "未分类"
source: "manual"
captured_at: "2026-04-30 10:34"
project: "默认"
status: "collected"
confidence: 0.5
---

> 这里写一句话总结。

# 这里写标题

这里写正文。
```

## 手动维护提醒

- 改标题时，同步改 `title`、一级标题和文件名。
- 标签可以多个，但不要制造同义重复。
- 分类只写一个。
- 项目不确定就写 `默认`。
- 状态不确定就写 `collected`。
