# 默认 Markdown 模板

适合普通 PinMind 输出。默认不展示原始内容。

```markdown
---
schema_version: "{{schema_version}}"
title: "{{title}}"
summary: "{{summary}}"
tags: {{tags}}
category: "{{category}}"
source: "{{source}}"
captured_at: "{{captured_at}}"
project: "{{project}}"
status: "{{status}}"
confidence: {{confidence}}
---

> {{summary}}

# {{title}}

{{body}}
```
