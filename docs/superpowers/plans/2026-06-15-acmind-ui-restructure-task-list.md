# AcMind 页面内容地图

> 本文档只说明 AcMind 每个页面里有什么。  
> 用于逐页交给 ChatGPT 生成理想状态 UI 图，再反推设计规范、排版、比例和尺寸。

## 1. 统一应用外壳

| 区域 | 内容 |
|---|---|
| 设计画布 | macOS 桌面窗口，建议统一使用 `1500 × 920` |
| 窗口控制 | 左上角红、黄、绿窗口按钮 |
| 产品区 | AcMind、产品地图、主导航状态、快捷键提示 |
| 核心工作流 | 首页、Agent、剪贴板 & 手机同步、日程、工具台 |
| 随身入口 | 灵动大陆 & 配置、说入法 |
| 系统与设置 | 状态、设置、模型管理 |
| 导航项 | 图标、名称、选中状态、快捷键 |
| 整体风格 | 浅色、中性、Apple 原生；不使用大面积彩色背景或营销页渐变 |

---

## 2. 首页

**页面作用：** 展示当前工作状态、系统状态和最常用入口。

| 区域 | 内容 |
|---|---|
| 顶部 | 问候语、简短副标题、快速记录、说入法、搜索 |
| 工作台总览 | 当前就绪状态、当前重点、当前页面、窗口状态、采样状态、最近刷新时间 |
| 主要入口 | Agent、剪贴板 & 手机同步、日程、状态 |
| 运行概览 | CPU、内存、网络上下行、最近 60 秒趋势、进程摘要、运行提醒 |
| 设备状态 | CPU、内存、网络、电池、磁盘、温度、风扇、权限 |
| 快速操作 | 说入法、采集、收集箱、设置 |

---

## 3. Agent

**页面作用：** 提供类似 ChatGPT 的对话体验，并支持任务执行、工具调用和项目上下文。

| 区域 | 内容 |
|---|---|
| 顶部 | 当前会话标题、当前项目、待命/忙碌、录音状态、当前模型、新建会话、侧栏开关 |
| 对话模式 | 普通对话、任务执行、Quick Ask、工具调用、自动化 |
| 最近记录 | 会话标题、会话状态、最近更新时间 |
| 项目上下文 | 当前项目、已加载文件或资料 |
| 对话流 | 系统欢迎消息、用户消息、Agent 回复、工具结果、任务进度、错误或权限提示、时间信息 |
| 输入区 | 多行输入框、发送、录音、附件、清空、当前模型、快捷建议 |
| 右侧面板 | 默认收起；展开后显示会话摘要、当前任务、最近结果、追溯信息 |

---

## 4. 收集箱

**页面作用：** 保存来自语音、截图、剪贴板和 Agent 的待整理内容。

| 区域 | 内容 |
|---|---|
| 顶部 | 页面标题、内容数量、搜索框 |
| 工作区 | 收集箱、剪贴板 & 手机同步 |
| 来源筛选 | 说入法、截图/OCR、剪贴板入箱、Agent 生成 |
| 状态筛选 | 待整理、已提炼、已归档、已导出 |
| 内容列表 | 标题、预览、来源、类型、时间、状态、标签、更多操作 |
| 空状态 | 说明内容会从语音、截图、剪贴板和 Agent 进入这里 |
| 右侧详情 | 全部数量、筛选数量、选中状态；选中后显示完整内容、来源、标签、提炼、归档、导出 |

---

## 5. 剪贴板 & 手机同步

**页面作用：** 管理近期复制内容、Pin 悬浮窗、粘贴队列和手机同步。

| 区域 | 内容 |
|---|---|
| 顶部 | 唯一页面标题、内容数量、收集箱切换、搜索、Pin 数、网格/列表切换、清空历史 |
| 类型筛选 | 全部、文本、链接、图片、文件、富文本、代码、视频 |
| 时间筛选 | 最近 24 小时、已收藏 |
| 实验功能 | 手机同步 |
| 内容卡片 | 类型、内容预览、来源应用、创建时间、Pin、收藏、复制、更多操作 |
| 卡片类型 | 文本、图片、文件、链接、富文本、代码、视频 |
| 右侧概览 | 总计数量、当前显示数量、Pin 数量 |
| Pin 管理 | 显示全部、隐藏全部、关闭全部 |
| 粘贴队列 | 队列内容、当前顺序、队列操作 |

## 7. 工具台

**页面作用：** 选择工具、配置运行参数并查看结果。

| 区域 | 内容 |
|---|---|
| 顶部 | 页面标题、简短流程说明、工具搜索、搜索快捷键 |
| 工作流 | 选择工具、配置运行、查看结果 |
| 工具概览 | 工具总数、最近使用数量、当前分类 |
| 分类筛选 | 全部、文本处理、内容转换、下载工具、AI 工具、开发工具、实用工具 |
| 工具卡片 | 图标、名称、说明、分类、标签、本地运行状态、打开/运行入口 |

| 工具详情 | 名称、说明、分类、标签、运行方式、配置参数、打开/重新运行、最近结果 |

| 区域 | 内容 |
|---|---|
| 顶部 | 页面标题、副标题“伴随能力的配置中心” |

| 右侧状态 | 权限摘要、热区参数、收起宽度、HUD 状态 |

## 10. 状态

**页面作用：** 展示系统资源、硬件传感器、运行趋势、诊断和权限状态。

| 区域 | 内容 |
|---|---|
| 顶部 | 页面标题、最近更新时间、采样状态、刷新 |
| 系统摘要 | CPU、内存、网络、电池、磁盘 |
| 硬件传感器 | 温度、风扇、GPU、热节流 |
| 采样状态 | 刷新状态、更新时间、是否采样 |
| 磁盘设备 | 设备数量、当前设备、数据来源、I/O 状态 |
| 健康摘要 | CPU 当前值、内存当前值、网络上下行、磁盘占用、电池或热状态 |
| 趋势区 | 最近 60 秒趋势、CPU、内存、网络、磁盘 |
| 诊断区 | 网络详情、电源详情、权限详情、设备温度、风扇转速 |
| 权限与能力 | 已授权、不可用、未知状态、不可用原因 |

## 13. UI 图生成约束

- 统一使用 `1500 × 920` 的 macOS 桌面窗口
- 保留统一主侧边栏和 macOS 窗口控制
- 使用浅色 Apple 原生风格
- 主色以中性白、浅灰和系统蓝为主
- 颜色只用于选中、状态、警告和成功
- 中间主内容区始终获得最大空间
- 右侧栏只展示高价值摘要或详情
- 不重复页面标题或摘要内容
- 不使用大面积无功能空白
- 不使用高饱和渐变背景
- 不设计成网页后台或营销页面
