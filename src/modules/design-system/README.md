# Design System 模块

## 职责

- 视觉 Token（颜色、字体、间距）
- 基础组件（Button、Input、Card 等）
- 图标系统
- 布局组件
- 主题切换

## 现有代码映射

- `src/renderer/design-system/` — 设计系统
  - `tokens.ts` — 视觉 Token
  - `primitives.tsx` — 基础组件
  - `icons.tsx` — 图标
  - `components/` — 组件库
  - `iconAssetMap.ts` — 图标资源映射
- `src/assets/icon/` — SVG 图标资源
